{% trans 'Privacy policy' %}

{% trans 'By using the ‘Send money to someone in prison’ service you agree to this privacy policy and to our terms and conditions.' %} {% trans 'Please read them carefully.' %}

{% trans 'Purpose' %}

{% trans 'This privacy policy sets out the standards that you can expect from Her Majesty’s Prison and Probation Service (HMPPS) when we request or hold personal information (‘personal data’) about you; how you can get access to a copy of your personal data; and what you can do if you think the standards are not being met.' %}

{% trans 'HMPPS is an Executive Agency/Arm’s Length Body of the Ministry of Justice (MoJ).' %} {% trans 'MoJ is the data controller for the personal information we hold.' %} {% trans 'HMPPS collects and processes personal data for the exercise of its own and associated public functions.' %} {% trans 'These include: providing safe and secure prison services, including those related to the management and rehabilitation of offenders.' %}

{% trans 'About personal information' %}

{% trans 'Personal data is information about you as an individual. It can be your name, address or telephone number.' %} {% trans 'It can also include information about the payments you make and the prisoner you are paying.' %}

{% trans 'We know how important it is to protect customers’ privacy and to comply with data protection laws.' %} {% trans 'We will safeguard your personal data and will only disclose it where it is lawful to do so, or with your consent.' %}

{% trans 'Types of personal data we process' %}

{% trans 'We only process personal data that is relevant for the services we are providing to you.' %} {% trans 'If you make a payment, this includes:' %}

  • {% trans 'your name, postal address, email address, payment card details, your IP address and details about your contact in prison' %}

{% trans 'If you provide feedback to us, this includes:' %}

  • {% trans 'your email address or other contact details you give us' %}
  • {% trans 'any personal information you include in questions, queries or feedback you leave' %}

{% trans 'Information about how you use the site is collected using cookies and page tagging techniques (Google Analytics).' %} {% url 'cookies' as link %} {% blocktrans trimmed %} It does not contain any personal data and is only collected if you consent to this. {% endblocktrans %} {% trans 'It is shared with the Government Digital Service.' %}

{% trans 'Purpose of processing and the lawful basis for the process' %}

{% trans 'We collect personal data so that you can safely send money digitally to someone in prison.' %} {% trans 'If we don’t have all the personal data we ask for, the payment can not be processed.' %}

{% trans 'It helps us to:' %}

  • {% trans 'complete the payment transactions' %}
  • {% trans 'prevent, detect and investigate suspicious payment' %}
  • {% trans 'prevent, detect and investigate suspected crime or unlawful activity' %}
  • {% trans 'respond to your feedback if you have asked us to' %}

{% trans 'The data you give us will be monitored to prevent and detect financial fraud, money laundering and other criminal activity.' %}

{% trans 'The data is processed in accordance with the General Data Protection Regulation and Data Protection Act 2018 and the lawful basis for processing is substantial public interest.' %}

{% trans 'Who the information may be shared with' %}

{% trans 'We sometimes need to share the personal information we process with the individual themselves and also with other organisations.' %} {% trans 'Where this is necessary we will comply with all aspects of the data laws.' %} {% trans 'The organisations we share your personal information with are:' %}

  • {% blocktrans trimmed with gov_url='https://www.payments.service.gov.uk/' wp_url='https://www.worldpay.com' %} GOV.UK Pay and WorldPay if the payment is made by debit card. {% endblocktrans %} {% trans 'This facilitates making a payment.' %} {% blocktrans trimmed with url='https://www.payments.service.gov.uk/privacy' %} See the privacy policy. {% endblocktrans %}
  • {% blocktrans trimmed with url='https://www.zendesk.com/' %} Zendesk to handle any feedback that you leave. {% endblocktrans %} {% blocktrans trimmed with url='https://www.zendesk.com/company/terms/' %} See the terms & conditions attached to this service. {% endblocktrans %}
  • {% blocktrans trimmed with url='https://www.mailgun.com/' %} Mailgun to send emails. {% endblocktrans %} {% blocktrans trimmed with url='https://www.mailgun.com/privacy-policy' %} See the privacy policy. {% endblocktrans %}
  • {% blocktrans trimmed with url='https://www.justice.gov.uk/about/hmps/contracted-out' %} Selected individuals employed in contracted-out prisons. {% endblocktrans %}

{% trans 'We may also share your data with the police and other law enforcement agencies where criminal or unlawful activity is suspected.' %}

{% trans 'Details of transfers to third country and safeguards' %}

{% trans 'Data collected is stored within the UK and Ireland.' %} {% trans 'However, due to the global nature of the internet, data may transit through other countries outside of the European Economic Area (EEA) while going to and from the database.' %} {% trans 'Data is not stored in a non-EEA country.' %}

{% trans 'Feedback about the service may be stored outside of the EEA, where it could be viewed by our staff or suppliers.' %} {% trans 'By submitting your personal data, you agree to this.' %}

{% trans 'Retention period for information collected' %}

{% blocktrans trimmed with url='https://www.justice.gov.uk/downloads/offenders/psipso/psi-2018/psi-04-2018-records-information-management-retention-policy.pdf' %} Data is retained for seven years in accordance with HM Prison and Probation Service retention policy. {% endblocktrans %}

{% trans 'Access to personal information' %}

{% trans 'You can find out if we hold any personal data about you by making a ‘subject access request’.' %} {% blocktrans trimmed with url='https://www.gov.uk/government/organisations/ministry-of-justice/about/personal-information-charter' %} This process is set out in the Ministry of Justice personal information charter. {% endblocktrans %}

{% trans 'When we ask you for personal data' %}

{% trans 'We promise to inform you why we need your personal data and ask only for the personal data we need and not collect information that is irrelevant or excessive;' %}

  • {% trans 'You can withdraw consent at any time, where relevant;' %}
  • {% trans 'You can lodge a complaint with the supervisory authority;' %}
  • {% trans 'We will protect it and make sure no unauthorised person has access to it;' %}
  • {% trans 'Only where appropriate and necessary share it with other organisations for legitimate purposes;' %}
  • {% trans 'Make sure we don’t keep it longer than is necessary;' %}
  • {% trans 'Not make your personal data available for commercial use without your consent; and' %}
  • {% trans 'Consider your request to correct, stop processing or erase your personal data.' %}
