{% trans 'About personal information' %}

{% trans 'Personal data is information about you as an individual. It can be your name, address, telephone number or email address.' %} {% trans 'It can also be information about the person you have contact with in prison.' %}

{% trans 'We know how important it is to protect your privacy and to comply with data protection laws.' %} {% trans 'We will keep your personal data safe and only disclose it when lawful to do so, or with your consent.' %}

{% trans 'Types of personal data we process' %}

{% trans 'We only process personal data relevant for the services we give you.' %} {% trans 'This is information about how you use the site to allow us to see how the service is being used so we can improve it.' %} {% trans 'It does not contain any personal data, unless you send a payment or feedback is submitted with an email address.' %}

{% trans 'It includes:' %}

  • {% trans 'your name, postal address, email address, bank details, your IP address and details about your contact in prison' %}
  • {% trans 'questions, queries or feedback you leave' %}
  • {% trans 'information on how you use the site, using cookies and page tagging techniques (Google Analytics) to help us improve it' %}

{% trans 'This helps us to:' %}

  • {% trans 'complete the transaction' %}
  • {% trans 'improve the site by monitoring how you use it' %}
  • {% trans 'respond to feedback, if you’ve asked for some' %}

{% trans 'Purpose of processing and the lawful basis for the process' %}

{% trans 'We collect personal data so that you can safely send money digitally to someone in prison.' %} {% trans 'If we don’t have all the personal data we ask for, the bank transfer or direct debit can not be processed.' %}

{% trans 'Data given will be monitored to protect from financial fraud.' %}
