{% trans 'Help with sending money to a prisoner' %}

  • {% blocktrans trimmed %} If you read this help section but still have a question about this service, contact us. {% endblocktrans %}

{% blocktrans trimmed %} This service is free and secure, and available in all public prisons in England and Wales. {% endblocktrans %} {% blocktrans trimmed with url='https://www.justice.gov.uk/about/hmps/contracted-out' %} But it is not available in private prisons. {% endblocktrans %}

{% trans 'You need a bank account to use this service.' %} {% blocktrans trimmed with url='https://www.gov.uk/staying-in-touch-with-someone-in-prison/sending-money' %} But you can send money to a prisoner using a postal order, a cheque or cash. {% endblocktrans %}

{% trans 'Why does it say no prisoner matches the details I enter?' %}

{% blocktrans trimmed %} This service is available in all public prisons in England and Wales. {% endblocktrans %} {% trans 'It normally takes one working day to get any new prisoner into the system and onto the database.' %}

{% trans 'Sometimes the prisoner’s date of birth and/or prison number are incorrectly entered into the prison database.' %} {% trans 'Please ask the prisoner to check with the wing officer what details are on the prison database to make sure your details match theirs.' %}

{% trans 'How long does it take?' %} {% trans 'Does the prisoner get told when they get their money?' %}

{% trans 'Money should reach the prisoner’s account in 1 working day, but your bank can take up to 3 working days to process the payment.' %} {% trans 'No money is processed at weekends or bank holidays.' %}

{% trans 'Prisoners receive a break-down receipt for money sent to them only when they collect their canteen.' %}

{% trans 'Help with bank transfers' %}

{% trans 'Do I need a new reference each time I send money by bank transfer?' %}

{% trans 'No, use the reference you first generated each time you send money to this particular prisoner.' %}

{% trans 'Why didn’t I get email confirmation after I made a bank transfer?' %}
