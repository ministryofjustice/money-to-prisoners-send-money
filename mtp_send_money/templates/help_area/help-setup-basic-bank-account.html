{% trans 'You can use the ‘Send money to someone in prison’ service if you have a debit card.' %} {% trans 'You can get a debit card with most bank accounts.' %}

{% trans 'If you don’t have a bank account, or you don’t think you’d get one, you might be surprised to know there is a basic, fee-free bank account for people:' %}

  • {% trans 'who don’t already have a bank account but want to give and receive money online' %}
  • {% trans 'who don’t qualify for a standard current account' %}
  • {% trans 'with poor credit history' %}
  • {% trans 'with no credit history' %}
  • {% trans 'who don’t have obvious proof of ID (for example, passport, driving licence)' %}
  • {% trans 'who don’t have obvious proof of address (for example, utility bills)' %}
  • {% trans 'who want to apply in person or by phone' %}
  • {% trans 'who don’t want to pay to have a bank account' %}

{% trans 'No proof of ID or address' %}

{% trans 'Most banks will accept:' %}

  • {% trans 'As proof of ID' %} – {% trans 'a letter from your GP / social landlord / vicar / care home manager / warden of sheltered accommodation, hostel or refuge / armed services officer' %}
  • {% trans 'As proof of address' %} – {% trans 'a letter from DWP / benefits / state pension / HMRC / JobCentre+ / your local council / a Blue Badge (disability) / your employer, college or training provider' %}

{% trans 'What to do next' %}

{% blocktrans trimmed with link='https://www.citizensadvice.org.uk/' %} Go to your local bank or Citizens Advice and they will help you set up your basic bank account. {% endblocktrans %} {% trans 'Ask them to go to this page on GOV.UK, if it helps.' %}
